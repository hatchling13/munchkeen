# munchkeen

[![pnpm](https://img.shields.io/badge/maintained%20with-pnpm-cc00ff.svg?style=for-the-badge&logo=pnpm)](https://pnpm.io/)

`munchkeen` is a Solid-first interactive graph exploration toolkit.

In v1, Cytoscape.js is the first renderer adapter and the first production renderer, but the library is architected around a renderer-neutral graph, projection, layout, and scene pipeline.

## Quick start

Install it:

```bash
npm i munchkeen
# or
yarn add munchkeen
# or
pnpm add munchkeen
```
